# Datalayer Experiments RTC

```bash
git clone https://github.com/datalayer-experiments/rtc.git && \
  git clone externals/automerge
git submodule init
git submodule update

yarn && \ # Install node modules.
  yarn build && \ # Build.
  yarn textarea # Start the textarea example.
```
